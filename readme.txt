== Installation ==

1. First, download the Auto Social Backlink Builder free plugin
2. Login to your WordPress Dashboard.
3. Navigate to "Plugins" > "Add New".
4. Then click "Upload" in the header.
5. Choose the zip file you downloaded earlier.
6. Click "Install Now".
7. Once installed, tap on "Activate Plugin"
8. Navigate to the "Settings" > "Auto Social Backlinks Free" to be show the settings screen.
